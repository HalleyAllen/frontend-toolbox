# 前端工具箱项目

## 项目简介
本项目是一个前端工具箱，集成了多种常用的工具功能，为开发者和普通用户提供便捷的在线工具服务。

## 功能需求

### 核心功能模块

#### 1. 常用工具
- **JSON格式化**：格式化、压缩JSON数据
- **时间戳转换**：Unix时间戳与日期时间互转
- **颜色工具**：颜色代码转换（HEX、RGB、HSL）
- **Base64编码解码**：文本与Base64互转
- **URL编码解码**：URL参数的编码与解码

#### 2. 文本工具
- **文本对比**：比较两段文本的差异
- **文本统计**：字数、字符数、行数统计
- **文本替换**：批量替换文本内容
- **正则表达式测试**：测试正则表达式匹配效果

#### 3. 表单工具
- **密码生成器**：生成强密码
- **随机数生成器**：生成指定范围的随机数
- **UUID生成器**：生成UUID
- **校验工具**：邮箱、手机号、身份证号等校验

#### 4. 转换工具
- **单位转换**：长度、重量、面积等单位转换
- **进制转换**：二进制、十进制、十六进制互转
- **货币转换**：不同货币之间的汇率转换

### 技术栈
- **前端框架**：React
- **构建工具**：Vite
- **样式方案**：CSS3 / 可选集成Tailwind CSS
- **状态管理**：React Context API（轻量级）
- **路由管理**：React Router
- **打包部署**：Vite构建，支持静态部署

## 特色功能
- **模块化设计**：每个工具独立封装，便于维护和扩展
- **响应式布局**：适配不同屏幕尺寸
- **实时预览**：工具操作实时反馈结果
- **本地存储**：使用localStorage保存用户设置和历史记录
- **主题切换**：支持浅色/深色模式

## 开发命令
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 部署方案
